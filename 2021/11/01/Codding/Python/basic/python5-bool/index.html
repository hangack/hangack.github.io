자료형의 참과 거짓

문자형이나 리스트, 튜플 등은 비어 있으면 거짓(False)가 되고, 비어 있지 않으면 참(True)이 된다. 숫자에서는 그 값이 0이 될 때 거짓이 된다.

[문자열]

"python" = Ture
"" = False

[리스트]

[1,2,3] = Ture
[] = False

[튜플]

(1,2,3) = Ture 
() = False

[딕셔너리]

{1,2,3} = Ture 
{} = False

[숫자형]

0이 아닌 수 = Ture 
0 = False
